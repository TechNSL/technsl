## Python Install

- Install Python from [https://python.org/](https://www.python.org/)
- Install Visual Studio Code from [https://code.visualstudio.com/](https://code.visualstudio.com/)
- Install Python extension in VS Code from [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-python.python)

## Check Python Installation

```sh
python --version
```

```sh
python -V
```

## Python Command Line

```sh
C:\Users\Your Name>python
```

```sh
C:\Users\Your Name>py
```
